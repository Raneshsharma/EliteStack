## 3. What's New

### 3.1 Template Parity: Add Education Descriptions to Modern and Minimal

**Issue:** `resume_classic.html` renders education descriptions (lines 92-100). `resume_modern.html` and `resume_minimal.html` do not.

**Fix in `resume_modern.html`:**
After the education degree line (after line 68: `{{ edu.degree }}...GPA...`), add:

```html
                {% if edu.description %}
                <ul style="margin-top:0.25rem; margin-left:1rem; font-size:0.875rem; color:#334155; list-style-type:disc;">
                    {% for line in edu.description.splitlines %}{% if line.strip %}
                    <li style="margin-bottom:0.125rem;">{{ line.strip }}</li>{% endif %}{% endfor %}
                </ul>
                {% endif %}
```

**Fix in `resume_minimal.html`:**
After the education degree line (after line 68: `{{ edu.degree }}...GPA}`), add:

```html
            {% if edu.description %}
                <p style="font-size:0.85rem; color:#555; margin-top:0.125rem;">{{ edu.description }}</p>
            {% endif %}
```

### 3.2 Print CSS: Add `break-inside: avoid` Rules

**Problem:** Without page-break rules, individual job entries can split across two pages, which looks unprofessional.

**Fix in all three templates:**

Replace the existing `@media print` block in each template with:

```html
<style>
@media print {
    @page { margin: 0.5in; size: letter; }
    body { background: white !important; -webkit-print-color-adjust: exact; print-color-adjust: exact; }
    nav, .print\:hidden, header nav, footer { display: none !important; }
    a { color: inherit !important; text-decoration: none !important; }

    /* Page break rules */
    h2 { break-after: avoid; page-break-after: avoid; }
    .resume-entry { break-inside: avoid; page-break-inside: avoid; }
    section { break-inside: avoid; page-break-inside: avoid; }

    /* Classic-specific */
    .border-t-2 { border-color: #888 !important; }

    /* Remove shadow/padding from outer containers */
    .resume-page { box-shadow: none !important; padding: 0 !important; border: none !important; margin: 0 !important; max-width: 100% !important; }
}
</style>
```

**Note:** The outer container class in each template differs. In `resume_classic.html`, the container div has `class="bg-white p-8 resume-content"`. The other templates use inline styles. Add `class="resume-page"` to the outer div in each template, then the `.resume-page` selector works consistently.

**Classic template:** Add `resume-page` to line 6:
```html
<div class="bg-white p-8 resume-content resume-page" ...>
```

**Modern template:** Add `class="resume-page"` to line 6:
```html
<div style="...;" class="resume-page">
```

**Minimal template:** Add `class="resume-page"` to line 6:
```html
<div style="...;" class="resume-page">
```
