#### Subtask 1A: Fix `resume_modern.html`

Read `d:/EliteSttack/templates/resumes/resume_modern.html`.

**Step 1: Add `resume-page` class to outer container**

Find line 6:
```html
<div style="...font-size: 10.5pt; line-height: 1.5; color: #1a1a2e;">
```
Change it to:
```html
<div style="font-family: 'Segoe UI', Arial, sans-serif; font-size: 10.5pt; line-height: 1.5; color: #1a1a2e;" class="resume-page">
```

**Step 2: Add education description block**

Find the education section (around lines 54-78). After this line (the degree/GPA line):
```html
<p style="font-size:0.875rem; font-style:italic; color:#475569;">{{ edu.degree }}{% if edu.field_of_study %} in {{ edu.field_of_study }}{% endif %}{% if edu.gpa %} · GPA: {{ edu.gpa }}{% endif %}</p>
```

Add this immediately after:
```html
                {% if edu.description %}
                <ul style="margin-top:0.25rem; margin-left:1rem; font-size:0.875rem; color:#334155; list-style-type:disc;">
                    {% for line in edu.description.splitlines %}{% if line.strip %}
                    <li style="margin-bottom:0.125rem;">{{ line.strip }}</li>{% endif %}{% endfor %}
                </ul>
                {% endif %}
```

**Step 3: Update `@media print` block**

Find the existing `<style>` block at the bottom (lines 123-128):
```html
<style>
@media print {
    body { background: white !important; }
    nav { display: none !important; }
}
</style>
```

Replace it entirely with:
```html
<style>
@media print {
    @page { margin: 0.5in; size: letter; }
    body { background: white !important; -webkit-print-color-adjust: exact; print-color-adjust: exact; }
    nav { display: none !important; }
    a { color: inherit !important; text-decoration: none !important; }
    h2 { break-after: avoid; page-break-after: avoid; }
    section { break-inside: avoid; page-break-inside: avoid; }
    .resume-page { box-shadow: none !important; padding: 0 !important; border: none !important; margin: 0 !important; max-width: 100% !important; }
}
</style>
```

    .resume-content { box-shadow: none !important; padding: 0 !important; border: none !important; margin: 0 !important; max-width: 100% !important; }
